## Run Locally

Clone the project

```bash
  git@github.com:dharmeshpal007/ghar-kharcha.git
```

Go to the project directory

```bash
  cd ghar-kharcha
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
